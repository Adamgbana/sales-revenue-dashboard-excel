# Sales & Revenue Analytics Dashboard (Excel)

## Project Overview

This project is an interactive Sales & Revenue Analytics Dashboard built in Microsoft Excel.

The dashboard transforms raw sales data into actionable business insights using Pivot Tables, Pivot Charts, and KPI reporting.

## Objectives

- Analyze revenue performance
- Identify top-performing regions
- Track sales representative performance
- Evaluate product and category revenue
- Support data-driven business decisions

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Dashboard Design

## Key Metrics

- Total Revenue
- Revenue by Region
- Revenue by Sales Representative
- Revenue by Category
- Revenue by Product

## Key Insights

- West region generated the highest revenue.
- Electronics was the top-performing category.
- John was the highest-performing sales representative.
- Revenue distribution was relatively balanced across regions.
